# ArvoreRB-EDII

Este projeto é uma implementação de uma **Árvore Rubro-Negra** desenvolvida como parte da disciplina **Estrutura de Dados II**.

## Objetivo
Praticar conceitos de árvores balanceadas e aplicar os princípios da Árvore Rubro-Negra em código.

## Conteúdo
- Inserção de nós
- Balanceamento automático
- Rotação à esquerda e à direita
- Alteração de cores
